算法训练 大等于n的最小完全平方数

2020-03-19 蓝桥杯 

/*
试题 算法训练 大等于n的最小完全平方数

资源限制
时间限制：1.0s   内存限制：256.0MB
问题描述
　　输出大等于n的最小的完全平方数。
　　若一个数能表示成某个自然数的平方的形式，则称这个数为完全平方数
　　Tips：注意数据范围
输入格式
　　一个整数n
输出格式
　　大等于n的最小的完全平方数
样例输入
71711
样例输出
71824
数据规模和约定
　　n是32位有符号整数
*/
#include <cmath>
#include <cstdio>
using namespace std;
int main() {
    long long n;
    scanf("%lld", &n);
    if (n < 0) {
        printf("%d", 0);
    }
    else {
        long long i = sqrt(n);
        while (i*i < n) {
			i++;				
		}
        printf("%lld", i * i);
    }
    return 0;
}
